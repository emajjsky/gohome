App 功能闭环

当前 V0 先用本机 edge-agent 跑通，后续再把事件和媒体同步到云端。

1. 接入摄像头

摄像头页只保存 RTSP / local 地址和账号信息；App 不直接连摄像头原始 RTSP。

2. 边缘端拉流

edge-agent 在本机拉流、抽帧、缓存截图；首页的实时预览走 720p MJPEG。

3. 输出检测事实

basic 输出亮度、对比度、运动分数；YOLO 模式再输出人数、人框和疑似跌倒候选。

4. 规则产事件

规则把检测事实转成离线、黑屏、无变化、无人、疑似跌倒等事件。

5. App 处理结果

App 首页看状态摘要，事件页看告警和截图，用户只做确认、误报、已处理。
